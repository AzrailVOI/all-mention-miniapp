# Скрипты для сборки и запуска

Этот проект включает скрипты для автоматической компиляции TypeScript и запуска приложения.

## Windows

### `build_and_run.cmd`
Полная сборка и запуск приложения:
- Устанавливает зависимости Node.js (если нужно)
- Компилирует TypeScript
- Запускает Flask приложение

**Использование:**
```cmd
build_and_run.cmd
```

### `build.cmd`
Только компиляция (без запуска):
- Устанавливает зависимости Node.js (если нужно)
- Компилирует TypeScript

**Использование:**
```cmd
build.cmd
```

## Linux/Mac

### `build_and_run.sh`
Полная сборка и запуск приложения:
- Устанавливает зависимости Node.js (если нужно)
- Компилирует TypeScript
- Запускает Flask приложение

**Использование:**
```bash
chmod +x build_and_run.sh
./build_and_run.sh
```

### `build.sh`
Только компиляция (без запуска):
- Устанавливает зависимости Node.js (если нужно)
- Компилирует TypeScript

**Использование:**
```bash
chmod +x build.sh
./build.sh
```

## Альтернативные способы

### Через npm
```bash
# Компиляция TypeScript
npm run build

# Компиляция и запуск
npm run dev

# Только запуск (после компиляции)
npm start
```

### Вручную
```bash
# Компиляция TypeScript
npx tsc --project tsconfig.json

# Запуск приложения
python main.py
```

## Требования

- **Node.js** (для компиляции TypeScript)
- **Python 3.8+** (для запуска приложения)
- **npm** (обычно устанавливается вместе с Node.js)

## Примечания

- Скрипты автоматически проверяют наличие необходимых инструментов
- При первом запуске будут установлены зависимости Node.js
- TypeScript компилируется в `webapp/static/js/`
- SCSS компилируется автоматически при запуске Flask приложения
